# AI Philosophy Club

Simple portfolio website for aiphilosophyclub.com

## About

This website serves as a personal hub for exploring the intersection of artificial intelligence and philosophical thought.

## Features

- Clean, responsive design
- Links to social media profiles (Substack, YouTube, LinkedIn, GitHub)
- About section introducing the site and its purpose

## Setup

Simply open `index.html` in a web browser to view the site locally.

## Deployment

To deploy this site:

1. Upload the files to your web hosting service
2. Point your domain (aiphilosophyclub.com) to the hosting service
3. Ensure `index.html` is set as the default page
